Bem-vindo ao projeto

O AGRINHO 2026 é um projeto voltado para inovação, agricultura e sustentabilidade, usando tecnologia para melhorar o futuro do campo.

Objetivos

  • Promover sustentabilidade no campo

  • Desenvolver soluções criativas
